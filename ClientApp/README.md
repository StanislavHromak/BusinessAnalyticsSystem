# Business Analytics System - Angular Client

Angular фронтенд для системи бізнес-аналітики.

## Встановлення

1. Встановіть Node.js 18+ та npm
2. Відкрийте термінал у папці `ClientApp`
3. Встановіть залежності:
```bash
npm install
```

## Запуск

```bash
npm start
```

Додаток буде доступний на http://localhost:4200

## Збірка для продакшену

```bash
npm run build
```

## Технології

- Angular 17
- TypeScript
- RxJS
- Standalone Components

## API

Клієнт підключений до ASP.NET Core Web API через proxy конфігурацію.
Переконайтеся, що API сервер запущений перед запуском Angular додатку.
